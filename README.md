# Intern React Portfolio

A simple React application built with Vite to demonstrate frontend development, Git workflow, pull requests, and AWS deployment preparation.

## Features

- Home page
- About page
- Contact page
- Responsive navbar
- Footer section
- Basic modern styling

## Tech Stack

- React
- Vite
- React Router DOM
- Git
- GitHub
- AWS S3
- AWS CodeBuild
- AWS CodePipeline

## Run Locally

```bash
npm install
npm run dev
